The Account Class

An abstract class that outlines the architecture for a checking or savings account.

Tutorial Account Checking Account Savings Account IAccountManager CheckingAccountManager SavingsAccountManager Ledger Reporter OverdraftException

Fields:

Field Description

accountID

A unique identifier for each account

Balance

The amount of money in an account

accountType

Describes the account as "checking" or "savings" type account

firstName

The first name of the account holder

lastName

The last name of the account holder
