Application Features: Learning with a Tutorial

Application Overview

This is a banking application, where users can create and view accounts, as well as withdraw and deposit funds. Follow along with the instructions on this page to familiarize yourself with the available features.

Tutorial

  1. Once the application is running, you will see the interface below.

    To begin, click "Create account," and create a checking account.

  2. You should see a printout of the contents of the new account.

  3. Next, to make a deposit, select the radio button adjacent to "Deposit" and enter a number.

  4. You should see your deposit reflected in the console.

  5. Next, select "Create account," and this time create a savings account.

  6. Make a withdrawal from the savings account of $2000.

  7. Select "View accounts," selecting a minimum balance of 30000 that will return the checking account.

    Voila, both accounts have balances that match the initial funds, with withdrawals and deposits included.
